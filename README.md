# AETHEREAL | Minimalist Sanctuary Hotel

A premium, minimalist hotel web experience designed as a frontend-focused project. This website showcases a modular design, atmospheric visuals (architecture and interiors), and a zero-JavaScript implementation for maximum performance and simplicity.

## Project Information

- **Developer**: Shivam Saini
- **Location**: Jaipur, India
- **University**: Poornima University
- **Project Type**: College Web Technology Project

## Design Philosophy

The project, "Aethereal," is built around the concept of "Simplicity is the Ultimate Luxury." It focuses on:
- **Warm Earth Palette**: Using cream, charcoal, and sage green tones.
- **Pure HTML & CSS**: Entirely functional without any script tags.
- **Immersive Visuals**: High-quality architectural shots focusing on hotel design and nature.
- **Responsive Layout**: A seamless experience across mobile, tablet, and desktop devices.

## Features

- **Home Page**: Atmospheric hero section, introduction to philosophy, and a curated experience grid.
- **The Hotel (Story)**: Narrative about the origin and architectural vision.
- **Suites**: Showcase of the Aethereal Suite and Horizon Loft.
- **Wellness**: Exploration of spa rituals and the infinity pool area.
- **Dining**: Insight into the "Culinary Altar" and farm-to-table philosophy.
- **Newsletter Subscription**: A minimalist CTA section for guest engagement.
